HiddenGems Travel App

HiddenGems is a smart travel app for discovering lesser-known destinations. It blends AI and user input to tailor trips to each explorer’s interests.

The design emphasizes simplicity and clarity, letting users focus on discovery while encouraging sustainable tourism through community integration.

Built using React and REST APIs, this app involved mapping, recommendation algorithms, and clean UX flows — turning every user’s journey into a personal adventure.
